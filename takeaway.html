본 과제를 수행하면서 느낀점


  처음 본 과제에 대한 내용을 들었을 때 작년에 웹 프로그래밍 과목을 수강하여서 꽤 자신이 있었습니다. 그래도 웹 프로그래밍에 대한 내용을 잊었을 것 같고 웹 프로그래밍 과목을 수강하면서 자신의 페이지를 만드는 것은 해보지 않았기 때문에 조금은 걱정이 됐습니다. 그런데 역시나 막상 과제를 직접 작업하려고 코딩을 시작하는 순간 이미 많은 내용을 잊었다는 것을 알게 되었습니다. 그래도 인터넷의 여러 예제 코드를 보면서 html css 등 웹 프로그래밍에 필요한 코드의 작동 원리를 조금 떠올릴 수 있었습니다. 하지만 단순히 html css 동작 원리를 안다고 해서 바로 개인 블로그를 나타낼 수 있는 웹 페이지를 제작하기가 어려웠고, 코드로 구현하기 전에 어떤 디자인으로 제작을 해야할지도 감이 잘 오지 않았습니다.

  그러던중 인터넷에서 Bootstrap 이라는 웹 페이지 제작에 많은 도움을 주는 오픈소스가 있다는 것을 알게되었습니다. Bootstrap 에서는 웹 페이지 제작에 필요한 Button, Navbar, Tooltip 등과 같은 컴포넌트의 코드를 제공해줄 뿐만 아니라 블로그, 앨범, 로그인 등의 웹 페이지 예시들의 코드 또한 제공합니다. 저는 이 Bootstrap 오픈 소스를 활용하여 보다 더 쉽고 간편하게 제 자신의 페이지를 제작할 수 있었습니다. 제가 생각하기에 해당 오픈 소스를 활용하지 않았다면 웹 페이지 제작에 더 많은 시간이 소요되고, 디자인 또한 덜 심미적일 것이라고 생각합니다. 그래서 이번 과제를 진행하면서 앞으로는 활용할 수 있는 여러 오픈 소스들에 대한 정보를 숙지하고 그것들을 활용하는 데에 능숙해져 제가 앞으로 할 작업의 효율을 더 높여야겠다는 생각이 들었습니다.

  아래는 제가 사용한 Bootstrap 이외 다른 웹 페이지 제작에 도움을 주는 오픈 소스입니다.

  1. webflow

  webflow는 개인 포트폴리오에서부터 식당, 의료, 피트니스 등의 비즈니스를 위한 페이지 템플릿까지 제공합니다.


  2. HTML5 UP

  HTML5 UP은 아주 유명한 웹 페이지 디자인 및 템플릿을 제공하는 사이트로 다양한 디자인의 웹 페이지 포맷에 관한 정보를 얻을 수 있습니다.


  3. colorlib

  colorlib은 wordpress 테마를 많이 제공하고 있어 블로그 관련 페이지를 작성하는데 많은 도움을 줍니다.


  마지막으로 github 사용 후 느낀 점에 대해 간단히 작성하겠습니다. 현업에서 github를 적극적으로 활용하여 여러 프로그램을 개발한다고 하였는데, 정확히 어떻게 어떤 매커니즘으로 이루어지는지 이해를 하지 못했습니다. 본 과제를 수행하면서 github를 어떻게 사용하여 인터넷에 저의 코드를 업로드하고 관리할 수 있는지 이해하게되었습니다. 이번에는 혼자 프로젝트를 하여서 다른 개발자와 코드를 공유하면서 작업을 할 수 있다는 git의 장점을 100% 활용할 수는 없었지만 다음에 또 다른 프로젝트를 수행하면서 협업의 툴로서 활용할 수 있을거라는 기대를 가지게 되었습니다.
